Explicação simples do CSS position

position: static (padrão)

Static

→ Não se move com top/left.

position: relative

Relative

→ Move, mas ainda ocupa seu espaço original.

position: absolute

Absolute

→ Sai do fluxo e fica preso ao elemento pai posicionado.

position: fixed

Eu fico fixo

→ Fica preso à tela mesmo rolando.
